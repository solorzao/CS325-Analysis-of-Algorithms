Program written in C++.

Compilation Notes: Uses ISO c++ 2011 standard (necessary for some C++ library functions to run). Compiled on OSU engineering server using -std=c++11 command.

Compile command: g++ -std=c++11 main.cpp -o rivalries

Run command: rivalries "file.txt"

Input will be read from file name specified in the command line. Replace "file.txt" with the name of the file you would like the program to read.
